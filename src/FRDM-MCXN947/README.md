# src/FRDM-MCXN947

NXP FRDM-MCXN947 の MCUXpresso SDK プロジェクトを配置するディレクトリ。

- 開発ボード: NXP FRDM-MCXN947（Cortex-M33 ×2 + DSP + eIQ Neutron NPU）
- 開発環境: MCUXpresso SDK（MCUXpresso IDE / VS Code 拡張）
- センサは I2C で直接接続（MLX90640 / VL53L5CX）
- 今回のスコープ: カラスの検出

セットアップ手順は [../../docs/firmware.md](../../docs/firmware.md) を参照。
