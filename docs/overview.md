# 概要 — 課題・ソリューション・全体アーキテクチャ

## 解決したい課題

都市部や住宅地では、カラスがゴミ袋を破り、ゴミ置き場を荒らす被害が発生している。従来の対策はネット、物理カバー、人による監視が中心であり、カラスの学習能力や設置環境によって効果が限定的になることがある。

カメラAIによる監視も考えられるが、住宅地では通行人や住民の映像を取得してしまうため、プライバシー面の課題がある。また、クラウドAIは通信環境、消費電力、運用コストの問題もある。

TerraGuard AI は、低解像度センサとオンデバイスAIを用いることで、カメラを使わず、クラウドにも依存しない害鳥・害獣検知を目指す。

---

## ソリューション概要

TerraGuard AI は、ゴミ置き場などの監視対象エリアに設置し、以下の情報を取得する。

- サーマルセンサによる温度分布
- ToF距離センサによる奥行き分布
- 背景との差分
- 時間方向の変化
- 変化領域の重心位置や移動量

本システムは、カラスをカメラ画像で直接認識するのではなく、ゴミ置き場に発生する状態変化を学習する。

分類対象の例：

```text
1. 通常状態
2. カラスによるゴミ漁り
3. 人によるゴミ出し
```

まずは「通常状態」と「カラスによるゴミ漁り」の2状態で確実に動作検証し、追加学習により3状態分類へ拡張する。

## 全体アーキテクチャ

DigiKey Make ONE Challenge 2026 版では、**NXP FRDM-MCXN947** を本番のAIボードとして、センサフュージョンとエッジAI処理による害鳥・害獣検知＋追い払いを実装する。

プロトタイプ段階のセンサ取得・可視化は、**Freenove Control Board V5（FNK0096 / Arduino UNO R4 WiFi 互換、メインMCU は Renesas RA4M1）** で行う。

```text
[プロトタイプ段階]
  MLX90640 / VL53L5CX
        │ I2C
        ▼
  Freenove Control Board V5 (RA4M1)
        │ シリアル(JSON)
        ▼
  PC（可視化・特徴量確認・データ収集）

            │ センサ・特徴量設計を移植
            ▼

[本番段階]
  MLX90640 / VL53L5CX
        │ I2C
        ▼
  NXP FRDM-MCXN947（センサフュージョン + エッジAI）
        │
        ▼
  追い払い機構（LED / ブザー / サーボ など）
```
