# 開発ロードマップ

## Step 1: プロトタイプ基板でのセンサ取得

- 対象ボード: Freenove Control Board V5 (RA4M1 / UNO R4 WiFi 互換)
- MLX90640 のサーマル画像取得（32×24）
- VL53L5CX の 8×8 距離データ取得
- PC 上で可視化（シリアル / JSON）

> 現状: LED点滅（Lチカ）で書き込み・シリアル疎通を確認済み。次はセンサ単体取得へ。

## Step 2: 差分データ生成

- 通常状態の背景データ取得
- サーマル差分マップ生成
- 距離差分マップ生成

## Step 3: ダミーカラス実験

- ヒーター入りダミーカラスを作成
- ゴミ袋付近で動かす
- 差分データが取れるか確認

## Step 4: 特徴量設計

- 平均変化量
- 最大変化量
- 重心位置
- 移動量
- 継続時間

## Step 5: 本番AIボードへの展開

- NXP **FRDM-MCXN947** へセンサ取得処理と特徴量を移植
- センサフュージョン + エッジAI推論を実装
- 通常状態 / カラスゴミ漁り / 人によるゴミ出しを分類

## Step 6: 追い払い機構の追加

- LED
- ブザー
- サーボ
- 音声
- 水噴射 など

検出から追い払いまでの一連の動作をデモとして仕上げる。
