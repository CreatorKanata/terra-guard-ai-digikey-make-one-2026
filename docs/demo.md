# デモ構成 — 仮想ゴミ置き場・実験手順

> **今回のスコープ: カラス検出のデモ。** FRDM-MCXN947 にセンサを直接接続し、カラスを検出できることを示す。

## 仮想ゴミ置き場

- ゴミ入りゴミ袋を 5〜6 個設置
- USBヒーターを 40℃ 程度に設定
- 鳥ぬいぐるみに USBヒーターまたはカイロを組み込む（ダミーカラス）
- ダミーカラスに棒を取り付け、センサ視野外から動かせるようにする

---

## デモ動作（カラス検出）

```text
1. 通常状態
   ゴミ袋だけを設置し、背景データを取得する（カラスなし）

2. カラスゴミ漁り状態
   ヒーター入りダミーカラスをゴミ袋付近で動かす

3. 検出結果表示
   FRDM-MCXN947 がカラスを検出したら、オンボードLED等で通知する
   （検証用にデバッグUART/MCU-Link経由でサーマル・距離・特徴量・判定を出力）
```

ポイント：

- 「通常状態 → カラス出現 → 検出」の流れを実演する
- 検出のオン/オフがダミーカラスの動きに追従することを示す

---

## 検証用の出力（開発時）

開発・チューニング時には、ボードから以下を確認できるようにする。FRDM-MCXN947 のデバッグUART（MCU-Link 経由の仮想COM）でPCへ送る想定。

- サーマルマップ / サーマル差分マップ
- 距離マップ / 距離差分マップ
- 抽出特徴量
- カラス検出の判定結果

これにより、センサが何を捉えているかを確認しながらしきい値やモデル入力を調整できる。
