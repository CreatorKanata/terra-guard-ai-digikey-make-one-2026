# デモ構成 — 仮想ゴミ置き場・PC表示・実験手順

## 仮想ゴミ置き場

- ゴミ入りゴミ袋を 5〜6 個設置
- USBヒーターを 40℃ 程度に設定
- 鳥ぬいぐるみに USBヒーターまたはカイロを組み込む（ダミーカラス）
- ダミーカラスに棒を取り付け、センサ視野外から動かせるようにする

---

## デモ動作

```text
1. 通常状態
   ゴミ袋だけを設置し、背景データを取得する

2. カラスゴミ漁り状態
   ヒーター入りダミーカラスをゴミ袋付近で動かす

3. 人によるゴミ出し状態
   人がゴミ袋を追加・移動する動作を行う

4. センサデータ表示
   PC上にサーマルマップ、距離マップ、差分マップを表示する

5. 判定結果表示
   LED、ブザー、サーボ、またはPC画面に状態分類結果を表示する
```

---

## PC表示項目

プロトタイプ段階では、AI判定だけでなく、センサデータを PC 上に表示する。これにより、センサが何を捉えているかを確認しながら AI 入力データを調整できる。

- サーマルマップ
- サーマル差分マップ
- 距離マップ
- 距離差分マップ
- 抽出特徴量
- 推論結果

ボードからは 115200 baud のシリアルで JSON 形式のデータを送信し、PC 側で受信・可視化する想定。

---

## 追い払い動作（本番）

検出結果に応じて、以下のいずれか／組み合わせで追い払い動作を行う。

- LED
- ブザー
- サーボ
- 音声
- 水噴射 など
